# Pourquoi ce livre existe

## Intention

Expliquer le besoin d'un manuel centré sur les systèmes plutôt que sur les annonces d'outils.

## Questions à traiter

- Pourquoi les systèmes d'IA échouent-ils souvent hors démonstration ?
- Quels invariants traversent les générations de modèles ?
- Comment raisonner au-delà des benchmarks isolés ?
