# Lab : RAG sur un dépôt de code

## Intention

Tester une chaîne de récupération sur un dépôt logiciel réel.

## Questions à traiter

- Quels documents extraire du dépôt ?
- Comment découper le code sans perdre la structure ?
- Comment évaluer la pertinence des réponses ?
