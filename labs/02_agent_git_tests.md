# Lab : agent Git et tests

## Intention

Explorer un agent capable de modifier du code et de vérifier ses changements.

## Questions à traiter

- Quelles commandes autoriser ?
- Comment limiter les modifications accidentelles ?
- Quels signaux de test suffisent pour fermer la boucle ?
